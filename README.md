# Swapnil Thakur — Portfolio

Built with React + Vite.

## Setup

```bash
npm install
npm run dev
```

## ⚠️ Important — Resume Link

Open `src/data/data.js` and replace `YOUR_RESUME_FILE_ID_HERE` in `resumeLink` with your actual Google Drive file ID.

To get it: upload your resume PDF to Google Drive → right-click → Share → Copy link. The ID is the long string between `/d/` and `/view` in the URL.

## Build

```bash
npm run build
```
